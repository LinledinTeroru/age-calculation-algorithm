# Advanced Age Calculation Algorithm

Profesyonel kariyer gelişimi için geliştirilmiş yenilikçi bir araç.

Bu algoritma, doğum tarihinizi temel alarak mevcut yaşınızı hassas bir şekilde hesaplar ve size anında motivasyonel geri bildirim sağlar.

## Özellikler

- Ultra hızlı yaş hesaplama (O(1) zaman karmaşıklığı)
- Gerçek zamanlı kariyer mindset güncellemesi
- Profesyonel hayatta yeni bir sayfa açma desteği
- Kullanıcı dostu ve basit arayüz (kod tabanlı)

## Kurulum ve Kullanım

### 1. Repoyu klonlayın

```bash
git clone https://github.com/LinledinTeroru/age-calculation-algorithm.git
cd age-calculation-algorithm
```

### 2. Kullanım

`age_calculator.py` dosyasını açın ve aşağıdaki kodu çalıştırın:

```python
# Doğum tarihinizi GG.AA.YYYY formatında girin
dogum_tarihi = "28.03.1990"

yas = 2026 - int(dogum_tarihi[-4:])

print(f"Tebrikler! Şu an tam {yas} yaşındasınız.")

# Motivasyonel geri bildirim
print("Bu algoritmayı çalıştırdığınız için bugün daha resilient, daha result-oriented ve daha optimistic bir profesyonel oldunuz.")
print("Kariyerinizde önemli bir adım attınız.")
```
